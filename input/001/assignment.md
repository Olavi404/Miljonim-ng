# JavaScripti kalkulaator

Loo lihtne veebipõhine kalkulaator, mis võimaldab kasutajal sisestada kaks arvu ja valida tehte.

## Nõuded

- Kasutaja saab sisestada kaks arvu.
- Kasutaja saab valida liitmise, lahutamise, korrutamise või jagamise.
- Nupule vajutades kuvatakse tulemus.
- Kui sisend pole arv, tuleb kuvada veateade.
- Nulliga jagamisel tuleb kuvada eraldi veateade.

## Hindamiskriteeriumid

- HTML sisaldab vajalikke sisestusvälju ja nuppu.
- JavaScript loeb kasutaja sisendi, teisendab selle arvuks ja kontrollib vigaseid olukordi.
- Tulemus kuvatakse kasutajale arusaadavalt.
